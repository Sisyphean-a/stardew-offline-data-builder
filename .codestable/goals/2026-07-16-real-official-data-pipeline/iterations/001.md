# Iteration 001

## 当前理解

原流程只接收夹具专用 `entries` 格式，真实 1.6.15 解包目录会解析为 0 条实体；本地 `stardew-valley-data` 的数组型 JSON 同样被忽略。真实官方数据存在对象模型、斜杠分隔旧格式和 LocalizedText 三类主要形态，且图片由精灵图/肖像路径与索引共同定位。

## 实现方案
