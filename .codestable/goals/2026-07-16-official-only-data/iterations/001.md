# Iteration 001

## Current Understanding

正式产物可以完全由官方解包数据构建。社区源过去只补充便利字段，不改变核心实体、
官方文本、搜索或图片，因此应删除整条社区运行链路，并从官方表之间推导可验证字段。

## Implementation Approach

- 删除社区 CLI 参数、加载器、匹配器、合并器、版本元数据、报告字段和测试夹具。
- 严格加载官方 `FishPondData`、`Locations`、`Machines`、`Shops` 等支持表。
- 建立官方物品索引和跨表引用，将作物、鱼类、NPC、商店、机器、鱼塘、配方等字段写入
  `officialDerived`，并保留官方来源信息。
- 对官方没有结构化等价物的攻略摘要和人工分类不做伪造。

## Changes This Iteration

- 正式 CLI 和构建流程不再接受 `--community-data`。
- 删除 `community_source`、社区 match/merge 管线及对应夹具。
- 数据库和 `.svdata` schema 升级为 v2，不再写入社区版本或 provenance。
- 新增官方支持数据加载、物品索引、字段归一化、商店引用和跨表 enrich。
- 更新 README、项目规范、设计文档和测试，使当前文档只描述官方数据流程。
